# 🐉 Dragoon News

A responsive real-time news web application built with React and Firebase Authentication.

## 🌐 Live Site
[gorgeous-meerkat-4cf3d2.netlify.app](https://gorgeous-meerkat-4cf3d2.netlify.app)

## ✨ Features
- 🔐 Email/Password Authentication
- 🔑 Google Login
- 📰 Real-time News by Category
- 🛡️ Protected Routes
- 📱 Fully Responsive

## 🛠️ Tech Stack
- React + Vite
- Firebase Authentication
- React Router v6
- Tailwind CSS + DaisyUI
- REST API Integration
